Hyper Text Markup Language(HTML) is the standard markup language for creating web pages and web applications.
Web browser receive HTML documents from a web server or local storage and render them into multimedia web pages.
HTML descirbes the structure of a web page semantically and originally included cues for the appearace of the documents.

HTML elements are the building blocks of HTML pages.
With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
It provides a means to create structured documents by detoning structural semanticx for text such as headings,paragraphs, lists, links, quotes and other items.
HTML elements are delineated by tags, wriiten using angle brackets.
